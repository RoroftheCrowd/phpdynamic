Instructions:

1. start up wamp server on your computer and enter localhost/phpmyadmin

2. create a database called hardware_sales

3. click on the database and go to import

4. place the folder into the www folder

5. go to google chrome and type in "localhost/phpdynamic"

6. Now the site is avablibe to use.
